CSCE 242: Web Applications

Zaniah Kirkpatrick

Learn how to effectively use visual studio code and github to create numerous web based projects/assignments.

Assignments

Assignment 1 - Basic HTML

Make page using basic html learned in class. Page must have 2 pictures, 2 paragraphs, a table, and a list of items.

Assignment 2 - Basic CSS

Make a page with 2 pictures, a background image, navigation links, and a table with borders.

Assignment 3 - Page Layout

Make a page layout using flexbox. Page should resemble sample layout given.
