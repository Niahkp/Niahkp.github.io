Assignments

Assignment 1 - Basic HTML

Make page using basic html learned in class. Page must have 2 pictures, 2 paragraphs, a table, and a list of items.

Assignment 2 - Basic CSS

Make a page with 2 pictures, a background image, navigation links, and a table with borders.

Assignment 3 - Page Layout

Make a page layout using flexbox. Page should resemble sample layout given.

Assignment 4 - Recreate USC

Recreate the UofSC home page.

Assignment 5 - Intro to Javascript

Create 3 colums that each perfrom different fucntions using javascript.

Assignment 6 - Conditionals

Use conditionals to create a color slider excercise and picture chooser excercise.

Assignment 7 - For Loops

Use for loops to draw infinite stars and tell which stars number was clicked.

Assignment 8 - Arrays

Use arrays to have pictures show messages when clicked.

Assignment 9 - Classes

Use classes to make a pop up box that shows info when clicked.

Assignment 10 - Json

sync to a json file and have it read the icecream name in an overlay
